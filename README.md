# Finance Academy - UI Kit & Component System

## Project Overview
This project is an online Finance School platform built using the **Atomic Design** methodology. It transitions from basic atoms (colors, typography) to complex molecules and organisms.

## Key Features
- **Bootswatch Integration**: Powered by the "Lux" theme for a professional, corporate financial aesthetic.
- **Custom UI Kit**: A dedicated section showcasing all design tokens.
- **Atomic Components**: Includes buttons, forms, cards, navigation, and auth organisms.
- **Responsive Layouts**: Optimized for financial data display and event management.

## Technical Stack
- HTML5 / CSS3
- Bootstrap 5 (Lux Theme)
- Google Fonts: Playfair Display & Inter

## How to explore
Open `index.html` to view the comprehensive UI Kit and the assembled organisms.
